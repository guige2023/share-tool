🔐 HTTPS 证书安装指南

ShareTool 使用自签名证书，首次访问时浏览器会显示安全警告。按以下步骤安装证书后可消除警告。

iOS（Safari）

1下载证书

在 Safari 中打开本页面，点击下方「下载证书」按钮，将证书保存到手机。

2安装证书

进入 设置 → 通用 → VPN 与设备管理，找到已下载的描述文件并安装。

3信任证书

进入 设置 → 通用 → 关于本机 → 证书信任设置，开启对 ShareTool 证书的完全信任。

Android（Chrome）

1下载并安装证书

在 Chrome 中打开本页面，点击下方「下载证书」按钮，然后用系统文件管理器打开安装。

2安装 CA 证书

系统会提示「为证书锁屏」——这是正常的。安装后 HTTPS 警告消除。

macOS（Safari / Chrome）

1下载并打开证书

点击下方「下载证书」按钮，在 Finder 中打开下载的证书文件。

2添加到钥匙串

证书会自动用「钥匙串访问」打开，选择「登录」钥匙串，点击「添加」。

3始终信任

在钥匙串中找到证书，右键「显示简介」→「信任」→「使用此证书时」→「始终信任」。

Windows（Chrome / Edge）

1下载证书

点击下方「下载证书」按钮，保存 .crt 文件。

2安装到受信任根证书

双击证书 →「安装证书」→「本地计算机」→「下一步」→「将所有证书放入下列存储」→「受信任的根证书颁发机构」→「确定」。

提示 如果以上步骤过于复杂，您也可以直接使用 HTTP 访问 ShareTool（部分功能如剪贴板自动同步可能受限）。局域网工具无需公网 HTTPS，证书主要用于消除浏览器警告。
